O que é o Dashboard Económico Portugal

O Dashboard Económico Portugal é um painel de dados macroeconómicos de acesso gratuito, construído com dados de fontes públicas e oficiais. O objectivo é centralizar os principais indicadores económicos do país — PIB, inflação, desemprego, dívida pública, Euribor e energia — numa interface clara e actualizada.

É um projecto independente, sem fins lucrativos, sustentado por publicidade (Google AdSense).
