Política de Privacidade
Dashboard Económico Portugal

Última actualização: Junho 2025

1. Quem somos

O Dashboard Económico Portugal é um site de informação e ferramentas financeiras, alojado em GitHub Pages, que disponibiliza dados macroeconómicos públicos de Portugal e ferramentas como o simulador de crédito à habitação baseado na Euribor.

2. Que dados recolhemos

Este site não recolhe dados pessoais directamente. Não existem formulários de registo, contas de utilizador nem bases de dados próprias com informação pessoal.

No entanto, os seguintes serviços de terceiros podem recolher dados técnicos de utilização:

  • Google AdSense — exibe anúncios publicitários e utiliza cookies para personalização de anúncios, mediante consentimento do utilizador.
  • GitHub Pages — aloja o site e pode registar logs de acesso técnicos (endereço IP, browser, timestamp).

3. Cookies

Este site utiliza dois tipos de cookies:

  • Cookies funcionais (necessários) — armazenamos os dados das APIs em localStorage do browser para melhorar a performance e garantir a disponibilidade dos dados mesmo quando as APIs estão temporariamente indisponíveis. Estes dados são armazenados localmente no seu dispositivo e não são transmitidos a terceiros.
  • Cookies de publicidade (opcionais) — se aceitar, o Google AdSense utiliza cookies para mostrar anúncios relevantes. Pode consultar a Política de Privacidade do Google para mais detalhes.

Pode gerir as suas preferências de cookies a qualquer momento através do banner de consentimento ou apagando os dados do site nas definições do seu browser.

4. Publicidade — Google AdSense

Este site utiliza o Google AdSense para exibir anúncios. O Google, enquanto fornecedor terceiro, utiliza cookies (incluindo o cookie DART) para exibir anúncios com base em visitas anteriores a este e a outros sites. Os utilizadores podem desactivar a utilização do cookie DART através da página de definições de anúncios do Google.

5. Fontes de dados

Os dados apresentados neste site são obtidos de fontes públicas:

  • BPstat / Banco de Portugal
  • INE — Instituto Nacional de Estatística
  • BCE — Banco Central Europeu
  • Eurostat
  • DGEG — Direção-Geral de Energia e Geologia
  • REN — Redes Energéticas Nacionais

Estes dados são de acesso público e são apresentados para fins informativos. Não constituem aconselhamento financeiro.

6. Os seus direitos (RGPD)

Ao abrigo do Regulamento Geral de Proteção de Dados (RGPD), tem o direito de aceder, rectificar ou apagar quaisquer dados pessoais que possamos ter sobre si. Uma vez que não recolhemos dados pessoais directamente, estes direitos aplicam-se principalmente aos dados recolhidos pelos serviços de terceiros referidos acima.

Para exercer os seus direitos relativamente a dados do Google, consulte o Google My Account.

7. Isenção de responsabilidade

As informações disponibilizadas neste site têm carácter meramente informativo e não constituem aconselhamento financeiro, fiscal ou jurídico. Os dados são obtidos de fontes públicas e podem conter imprecisões ou atrasos. Consulte sempre um profissional qualificado antes de tomar decisões financeiras.

8. Contacto

Para questões relacionadas com privacidade, pode contactar-nos através do repositório GitHub: github.com/tudofinance-lgtm/dashboard-economia-portugal.
